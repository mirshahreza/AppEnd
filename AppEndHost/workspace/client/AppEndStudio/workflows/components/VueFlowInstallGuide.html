Vue Flow Workflow Designer

Installation Required

Vue Flow packages are not installed. Please follow the steps below to install them.
1Open Terminal/Command Prompt

Navigate to your project directory

2Run Installation Command

Choose one of the following based on your package manager:

npm install @vue-flow/core @vue-flow/background @vue-flow/controls @vue-flow/minimap

Or with Yarn:

yarn add @vue-flow/core @vue-flow/background @vue-flow/controls @vue-flow/minimap
3Reload Application

After installation, refresh your browser to use the new designer

Features
  • Professional drag & drop interface
  • Visual workflow connections
  • Activity palette with categorized components
  • Properties panel for editing
  • Minimap and zoom controls
  • Auto-layout capabilities
  • 100% Free & Open Source (MIT License)
Pro Tip: Vue Flow is actively maintained with excellent documentation at vueflow.dev
