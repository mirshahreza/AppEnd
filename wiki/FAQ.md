# FAQ

## What is AppEnd?
A full?stack low?code/RAD platform for building APIs and UIs on .NET and SQL Server.

## Which OS and DB are supported?
Linux and Windows; Microsoft SQL Server for database.

## Which UI stack is used?
Vue.js 3 with Bootstrap.

## Can I add custom code?
Yes, both server?side and client?side code injection is supported.

## Is SSO supported?
OpenID Connect SSO is on the roadmap.

## How do I deploy?
Publish `AppEndHost`, configure settings, set up reverse proxy, run DB scripts. See [[Deployment]].

## Where can I ask questions?
Open an issue or start a discussion on GitHub.
