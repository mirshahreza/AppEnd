# Extensibility

Extend AppEnd on both server and client.

## Server?side
- Add services in `AppEndServer` and wire via DI in `AppEndHost`
- Implement pre/post CRUD hooks for validation and business rules
- Create custom endpoints for non?CRUD scenarios

## Client?side
- Build custom Vue components and pages
- Extend generated forms/lists with custom behaviors
- Add localization files and switch languages

## Packages
- Export/import modules as portable packages
- Version modules and document changes

See [[API Development]] and [[UI Development]].
