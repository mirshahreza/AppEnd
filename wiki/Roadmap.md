# Roadmap

This roadmap reflects high?level plans. Priorities may change.

## Phase 1: Application builder & CRUD (In progress)
- Responsive tables and lists
- New UI widgets for better create/update forms
- Advanced search bars for generated lists
- UI designer
- Docker image for easy installation
- Package Manager for portable plugins
- Git integration for production workflows
- OpenID Connect (SSO)
- Task Scheduler

## Phase 2: Workflow Engine (Planned)
- Visual workflow designer
- Workflow execution and monitoring
- Integration hooks for services and APIs

## Phase 3: Reporting & Visualization (Planned)
- Report designer
- Charts and dashboards
- Export options
